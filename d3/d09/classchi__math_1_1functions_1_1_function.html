Detailed Description

This is a base class for a complexity-limited function call. What we want to support is the following. i) Return of vector values (dim=1 to N). ii) Callable with a) Just a position b) Just the value of one or more variables c) With position AND and the value of one or more variables and the ability.

We can achieve this goal to some degree by defining the general function

\[ F(x,y,z,[\phi_0, \dots, \phi_{N^a}]) \mapsto \mathbb{R}^b \]

where \( N^a \) is the dimension of the input values (this excludes the position variables), and \( b \) is the number of output values.
