25 /** Sets a property of a Diffusion solver. Please also consult the whitepaper
26  * for the Diffusion solver (<a
27  * href="../../whitepages/DiffusionSolver/DiffusionSolver.pdf">
28  * Diffusion Whitepaper</a>)
29 
30 
31 
32 \param SolverHandle int Handle to an existing diffusion solver.
33 \param PropertyIndex int Code for a specific property.
34 \param Values varying Number of inputs associated with the index.<br>
35 
36 ##_
37 
38 ###PropertyIndex\n
39 DISCRETIZATION_METHOD\n
40  Discretization method. Expects to be followed by <B>DiscretizationMethod</B>
41  (see below).\n\n
42 
43 MAX_ITERS\n
44  Solver maximum number of iterations.\n\n
45 
46 RESIDUAL_TOL\n
47  Residual tolerance. Expects to be followed by a floating point
48  value (i.e. 1.0e-6).\n\n
49 
50 BOUNDARY_TYPE\n
51  Boundary type. Expects boundary index then <B>BoundaryTypeIndex</B>
52  then type value.\n\n
53 
54 
55 PROPERTY_D_MAP\n
56  Followed by an integer, sets the property index of a material
57  from where the solver will collect the diffusion coefficient.\n\n
58 
59 PROPERTY_Q_MAP\n
60  Followed by an integer, sets the property index of a material
61  from where the solver will collect the volumetric source value.\n\n
62 
63 PROPERTY_SIGMAA_MAP\n
64  Followed by an integer, sets the property index of a material
65  from where the solver will collect the absorbtion cross-section
66  \f$ \sigma_a \f$.\n\n
67 
68 \code
69 chiDiffusionSetProperty(solver,BOUNDARY_TYPE,2,DIFFUSION_DIRICHLET,1.0)
70 \endcode
71 
72 ###DiscretizationMethod\n
73  PWLC\n
74  Piecewise Linear Finite Element Continuous.\n\n
75 
76  PWLD_MIP\n
77  Piecewise Linear Finite Element Discontinuous using the Modified
78  Interior Penalty (MIP) method.\n\n
79 
80 
81 ### BoundaryTypeIndex
82 DIFFUSION_REFLECTING
83  Reflecting boundary conditions. Synonymous with Neumann with a
84  derivative of 0.0.
85  \f[ -D \hat{n}\cdot \nabla \phi = 0 \f]\n\n
86 
87 DIFFUSION_DIRICHLET\n
88  Constant value boundary condition.
89  Expects to be followed by a value \f$ f \f$ associated with \f$ \phi \f$.
90  \f[ \phi = f \f]\n\n
91 
92 DIFFUSION_NEUMANN\n
93  Constant derivative boundary condition. Expects to be followed
94  by a constant \f$ f \f$ representing
95  \f[ -D \hat{n}\cdot \nabla \phi = f \f]\n\n
96 
97 DIFFUSION_VACUUM\n
98  Vacuum boundary conditions. More appropriate to neutron diffusion.
99  \f[ \frac{1}{4}\phi + \frac{1}{2} D \hat{n}\cdot \nabla \phi = 0 \f]\n\n
100 
101 DIFFUSION_ROBIN\n
102  Robin boundary condition of the form
103  \f[ a \phi + b D \hat{n}\cdot \nabla \phi = f \f]\n\n
104 
105 \ingroup LuaDiffusion
106 \author Jan*/
