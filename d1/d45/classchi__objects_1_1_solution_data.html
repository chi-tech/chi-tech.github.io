Detailed Description

A solution data object allows one to establish a hierarchy of different data blocks of extensible types. In other codes the solution invariable means a single numerical-valued vector however this limits the notion of a solver producing "sets" of solutions, e.g., DiscreteOrdinates creating phi, psi and precursor solutions.
