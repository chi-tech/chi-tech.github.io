void chi_mesh::CellTriangle::FindBoundary2D ( chi_mesh::Region *  region)
virtual

This function operates on cells with an open edge and finds the boundary associated with such an edge. For a 2D operation it only tries to match to the line-mesh associated with the boundary. If a boundary was found then the edge index is set to >=0, i.e. e_index[e][3]=0
