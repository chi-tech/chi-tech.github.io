6 /** \page MeshTutorial_01 Mesh Tutorial 1: Predefined 2D Mesh Setup
7 
8 Consider an example of using a predeveloped mesh as shown in Figure 1 below.
9 
10 \image html Meshing/SquareMesh2x2.png "Figure 1 - Mesh spanning from [-1,-1] to [1,1]" width=200px
11 
12 
13 The first step in this process is to create a chi_mesh::MeshHandler for all
14 subsequent meshing operations. This will also serve as a global subspace for
15 other meshing objects that will be added.
16 
17 \code
18 chiMeshHandlerCreate()
19 \endcode
20 
21 We next import a surface mesh (chi_mesh::SurfaceMesh) that will define our
22  problem. **Solvers will directly operate
23 on predefined meshes**.
24 
25 \code
26 newSurfMesh = chiSurfaceMeshCreate();
27 chiSurfaceMeshImportFromOBJFile(newSurfMesh,"CHI_RESOURCES/TestObjects/SquareMesh2x2.obj")
28 \endcode
29 
30 The next step is to break up the boundaries of this mesh into edges that we can
31 assign to boundaries. We first collect *loops* of edges, which are lists of
32  connected edges. We then split these loops by angle into further loops. Finally
33  we assign each fine-grained loop to a chi_mesh::LineMesh object that can be
34  added to a chi_mesh::Boundary type.
35 
36  \code
37 line_mesh = {};
38 line_mesh_count = 0;
39 --
40 for k=1,loop_count do
41  split_loops,split_count = chiEdgeLoopSplitByAngle(loops,k-1);
42  for m=1,split_count do
43  line_mesh_count = line_mesh_count + 1;
44  line_mesh[line_mesh_count] = chiLineMeshCreateFromLoop(split_loops,m-1);
45  end
46 --
47 end
48 \endcode
49 
50 Next we create a chi_mesh::Region object to which we are going to add boundaries.
51 We can add a boundary for each chi_mesh::LineMesh and each chi_mesh::SurfaceMesh.
52 
53 \code
54 region1 = chiRegionCreate()
55 chiRegionAddSurfaceBoundary(region1,newSurfMesh);
56 for k=1,line_mesh_count do
57  chiRegionAddLineBoundary(region1,line_mesh[k]);
58 end
59 \endcode
60 
61 We now have everything we need to do a Predefined 2D mesh operation. To do this
62 we create a chi_mesh::SurfaceMesher with the type *SURFACEMESHER_PREDEFINED* as
63  well as a chi_mesh::VolumeMesher with the type *VOLUMEMESHER_PREDEFINED2D*.
64 
65 \code
66 chiSurfaceMesherCreate(SURFACEMESHER_PREDEFINED);
67 chiVolumeMesherCreate(VOLUMEMESHER_PREDEFINED2D);
68 --
69 chiSurfaceMesherExecute();
70 chiVolumeMesherExecute()
71 \endcode
72 
73 The *SURFACEMESHER_PREDEFINED* is just a pass-through mesher, in contrast to the
74  *SURFACEMESHER_DELAUNAY* type which will re-mesh surfaces. The
75  *VOLUMEMESHER_PREDEFINED2D* similarly does not create new mesh elements but at
76  least this instantiates the notion of nodes and cells.
77
