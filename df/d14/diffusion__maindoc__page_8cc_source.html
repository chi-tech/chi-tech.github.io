1 /** \defgroup LuaDiffusion Diffusion
2  * \ingroup LuaModules
3  *
4  *
5  * Please consult the whitepaper for this solver (<a
6  * href="../../whitepages/DiffusionSolver/DiffusionSolver.pdf">
7  * Diffusion Whitepaper</a>). This solver solves the general diffusion equation
8  * of the form
9  *
10  * \f{equation}{
11  * -\nabla D \nabla \phi + \sigma_a \phi = q.
12  * \f}
13  *
14  * Given a discretization method this solver will assemble the matrix \f$ A \f$
15  * using
16  * a system of linear equations. The solver can operate on 1D slabs, 2D polygon meshes
17  * and 3D polyhedron meshes. Currently two spatial discretization schemes are
18  * supported, Piecewise Linear Continous (PWLC) and
19  * Piecewise Linear Discontinous (PWLD) using the Modified Interior
20  * Penalty (MIP) method. The solver is fully parallel (using
21  * PETSc).
22  *
23  * Boundary conditions are specified by referencing unique boundary id's. i.e.
24  * assuming one uses the extruder mesher the template boundary would've been
25  * assigned to index 0 and this will be transfused to all the boundaries in the
26  * lateral periphery. The top and bottom boundaries are flat for extruded
27  * geometries and therefore are assigned their own id's. In the extruded case
28  * the last boundary index is always assigned to the top boundary and
29  * second-to-last boundary index is always assigned to the bottom boundary. Boundary
30  * types are specified using the ChiLua::chiDiffusionSetProperty function call,
31  * using the BOUNDARY_TYPE property index.
32  *
33  * The materials and source values are for now obtained from materials
34  * associated with cells. The default property mapping is shown below. If
35  * the property at index [1] (which is mapped to the source value q) is not
36  * available the source will default to a constant value of 1.0.
37  *
38  * \image html "DiffusionMatProp.png" width=500px
39  *
40  * To change the mapping of the properties the user needs to make a call to
41  * ChiLua::chiDiffusionSetProperty using the PROPERTY_D_MAP, PROPERTY_Q_MAP or
42  * PROPERTY_SIGMAA_MAP property index.
43  *
44  * By default the solver populates a scalar field function which is the solution
45  * \f$ \phi \f$.
46  *
47  * */
