Detailed Description

Cubic spline object.

This object has two internal vectors containing the knot interpolation points and function values. It also has a dynamically created matrix vector containing all the z values.

The spline can handle 4 types of boundaries:

  1. Free ends (second derivatives zero and both ends)
  2. Left fixed
  3. Right fixed
  4. Both ends fixed

For information on the math consult the whitepaper "Spline interpolation"
