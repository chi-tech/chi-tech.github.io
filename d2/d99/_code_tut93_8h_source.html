6\section CodeTut93Sec1 1 Introduction
7Monte Carlo based flux estimators traditionally use a number of tracks, traced
8within a volume, to estimate the scalar flux. For \f$ N_t \f$ amount of tracks
9traced inside a volume, \f$ V \f$, originating from a sample of \f$ N_p \f$
10number of particles/rays, the scalar flux, \f$ \phi \f$, can be estimated with
11\f[
12\phi \approx \frac{1}{N_p V} \sum_t^{N_t} \ell_t,
13\f]
14where \f$ \ell_t \f$ is the \f$ t \f$-th track length. This simply means that
15the scalar flux is the "average track length per unit volume".
16
17The individual tracks of these particles can be assigned a weight, \f$ w_t \f$,
18enabling a multitude of features.
19\f[
20\phi \approx \frac{1}{N_p V} \sum_t^{N_t} \ell_t w_t,
21\f]
22For our purposes we are interested in three possible
23weightings, i) weighting by a spherical harmonic, which can be used to compute
24flux moments, ii) weighting by average FE
25shape function values, allowing the projection of flux onto a FE space, and
26finally iii) weighting with an exponential attenuation, allowing the computation
27of uncollided flux.
28
29\subsection CodeTut93Sec1_1 1.1 Weighting with a spherical harmonic
30Applying a weight with a spherical harmonic is conceptually simple. For the
31\f$ t \f$-th track we simply set the weight to the spherical harmonic evaluated
32with the direction, \f$ \boldsymbol{\Omega}_t \f$, of the track,
33\f[
34\phi_{\ell m} \approx \frac{1}{N_p V} \sum_t^{N_t} \ell_t Y_{\ell m}(
35\boldsymbol{\Omega}_t),
36\f]
37
38\subsection CodeTut93Sec1_2 1.2 Weighting by average FE shape function values
39For a DFEM projection of the fluxes we require nodal values, on each cell, for
40a given flux quantity, \f$ \phi \f$. Therefore we seek \f$ \phi_h \f$ such that
41\f{eqnarray*}{
42\int_V b_i \phi_h dV = \int_V b_i \phi dV
43\f}
44and since \f$ \phi_h = \sum_j b_j \phi_j \f$ we need to solve the cell-by-cell
45system defined by
46\f{eqnarray*}{
47\sum_j \phi_j \int_V b_i b_j dV = \int_V b_i \phi dV.
48\f}
49In this system the entries \f$ \int_V b_i b_j dV \f$ are simply the entries of
50the mass matrix, \f$ M_{ij} = \int_V b_i b_j dV \f$, and we still need to find
51the rhs-entries \f$ \int_V b_i \phi dV \f$. This is where we will use the track
52length estimators by weighting with the shape functions b_i.
53
54
55In this formulation we have
56\f{eqnarray*}{
57\int_V b_i \phi dV \approx \frac{1}{N_p} \sum_t \ell_t w_t^{i,avg}
58\f}
59where
60\f[
61w_t^{i,avg} = \frac{ \int_{s_a}^{s_b} b_i(s\mapsto \mathbf{x}) ds }
62 {\int_{s_a}^{s_b} ds} =
63\frac{ \int_{s_a}^{s_b} b_i(s\mapsto \mathbf{x}) ds }
64 {\ell_t},
65\f]
66the average basis function value along the track. For ordinary Lagrange FE shape
67functions, which are not defined piecewise, the integral in the numerator can be
68obtained exactly using a numerical quadrature. For our applications, where we
69use the PWLD FE shape functions we need to split this integral per segment of
70the basic cell crossed.
71
72For example, consider the polygon below, where a ray is traced from position
73\f$ s_a \f$ to \f$ s_b \f$.
74\image html CodingTutorials/Tut93_segments.png width=350px
75
76The track traced across the cell needs to split into the segments defined by the
77sub-triangles of the polygon it crossed. Therefore the track \f$ s_a \to s_b \f$
78needs to be split into tracks \f$ s_0 \to s_1 \f$, \f$ s_1 \to s_2 \f$ and
79\f$ s_2 \to s_3 \f$ as per the figure. Therefore the integral becomes
80
81\f[
82\int_{s_a}^{s_b} b_i(s\mapsto \mathbf{x}) ds =
83\int_{s_0}^{s_1} b_i(s\mapsto \mathbf{x}) ds +
84\int_{s_1}^{s_2} b_i(s\mapsto \mathbf{x}) ds +
85\int_{s_2}^{s_3} b_i(s\mapsto \mathbf{x}) ds
86\f]
87which we can evaluate analytically since it is linear on each
88segment.
89
90**Note:** The mapping of \f$ s\mapsto \mathbf{x} \f$ can be quite expensive so
91in this particular case it would be better to evaluate the shape function at
92the half-way point of each segment, after which the integral becomes
93\f[
94\int_{s_a}^{s_b} b_i(s\mapsto \mathbf{x}) ds =
95b_i(\frac{s_0 + s_1}{2}\mapsto \mathbf{x}) (s_1 - s_0) +
96b_i(\frac{s_1 + s_2}{2}\mapsto \mathbf{x}) (s_2 - s_1) +
97b_i(\frac{s_2 + s_3}{2}\mapsto \mathbf{x}) (s_3 - s_2) +
98\f]

100\subsection CodeTut93Sec1_3 1.3 Weighting with an exponential attenuation
101Weighting with an exponential attenuation adds the final piece of weighting
102necessary to efficiently compute the uncollided flux.
103
104The exponential attenuation of the uncollided flux, \f$ \psi \f$, along the path
105of a ray with direction \f$ \boldsymbol{\Omega} \f$ is expressed as
106\f[
107\frac{d\psi}{ds} = -\sigma_t(s) \psi(s)
108\f]
109where \f$ s \f$ is the distance traveled and \f$ \sigma_t \f$ is the total
110cross section. From this model we can compute the attenuation across a cell,
111with constant \f$ \sigma_t \f$, as
112\f[
113\psi(s) = \psi(s=0) e^{-\sigma_t s}
114\f]
115where \f$ \psi(s=0) \f$ is the value of \f$ \psi \f$ when it entered the cell.
116
117
118To assimilate all of this into a raytracing algorithm we start a source particle
119with a weight, \f$ w^p = 1 \f$, which acts as the proxy for \f$ \psi \f$
120(i.e., \f$ w^p \equiv \psi \f$). From
121this we can determine the nodal uncollided flux, \f$ \phi^{uc} \f$, in a similar
122fashion as we would determine the regular flux, i.e.,
123\f[
124\sum_j \phi_j^{uc} \int_V b_i b_j dV = \int_V b_i \phi^{uc} dV,
125\f]
126however, now we need additional treatment for the integral containing
127\f$ \phi^{uc} \f$, for which we have
128\f[
129\int_V b_i \phi^{uc} dV \approx \frac{1}{N_p} \sum_t \ell_t w_t^{i,avg},
130\f]
131where, this time,
132\f[
133w_t^{i,avg} =
134\frac{ \int_{s_a}^{s_b} w^p(s) b_i(s\mapsto \mathbf{x}) ds }
135 {\ell_t}.
136\f]
137Note here that \f$ w^p \f$ is a function of position, specifically
138\f[
139w^p(s) = w^p(s=s_a) e^{-\sigma_t s}
140\f]
141and so is the basis function \f$ b_i \f$.
142
143The form of this integral needs to split into segments in the same way we
144did in the previous subsection. Therefore, given K amount of segments, we now have
145\f[
146\ell_t w_t^{i,avg} = \sum_{k=0}^{K-1} \ell_{tk} w_{tk}^{i,avg}
147\f]
148where \f$ \ell_{tk} \f$ is the track length of the \f$ k \f$-th segment and
149\f$ w_{tk}^{i,avg} \f$ is the average weight of this segment. The weight is
150computed with
151\f[
152w_{tk}^{i,avg} =
153\frac{ \int_{s_k}^{s_{k+1}} w^p(s) b_i(s\mapsto \mathbf{x}) ds }
154 {\ell_{tk}}.
155\f]
156where \f$ s_k \f$ and \f$ s_{k+1} \f$ are the beginning and ending positions of
157segment \f$ k \f$ respectively.
158
159Note here that, since we have an expression for \f$ w^p \f$, we can compute
160\f$ w^p \f$ at any point along track \f$ t \f$ including at the start of any
161segment. Therefore we define
162\f$
163w_k^p = w^p(s=s_k)
164\f$
165which allows us to express \f$ w^p \f$ as
166\f[
167w^p(s) = w_k^p e^{-\sigma_t(s-s_k)}, \quad \quad \quad s\in[s_k,s_{k+1}]
168\f]
169
170
171Additionally we express the basis functions on a segment, since we
172know the shape function is linear on the segment, as
173\f[
174b_i(s) =
175b_{i,k} \frac{s_{k+1}-s}{s_{k+1}-s_k} +
176b_{i,k+1} \frac{s-s_k}{s_{k+1}-s_k}
177\f]
178where \f$ b_{i,k} \f$ and \f$ b_{i,k+1} \f$ are the basis function values at the
179beginning and end of the segment, respectively. These two expressions allow us to
180evaluate the segment average weight as
181\f[
182w_{tk}^{i,avg} =
183\frac{1}{\ell_{tk}}
184\int_{s_k}^{s_{k+1}} w_k^p e^{-\sigma_t(s-s_k)}
185\biggr[
186b_{i,k} \frac{s_{k+1}-s}{s_{k+1}-s_k} +
187b_{i,k+1} \frac{s-s_k}{s_{k+1}-s_k}
188\biggr] ds
189\f]
190and since \f$ s_{k+1}-s_k = \ell_{tk} \f$ we can simplify this expression as
191\f[
192w_{tk}^{i,avg} =
193\frac{w_k^p}{\ell_{tk}^2}
194\int_{s_k}^{s_{k+1}} e^{-\sigma_t(s-s_k)}
195\biggr[
196C_0 + C_1 s
197\biggr] ds
198\f]
199where
200\f[
201C_0 = b_{i,k} s_{k+1} - b_{i,k+1} s_k
202\f]
203\f[
204C_1 = b_{i,k+1} - b_{i,k}.
205\f]
206
207With these constants defined the expression can be evaluated analytically
208\f[
209w_{tk}^{i,avg} =
210\frac{w_k^p}{\ell_{tk}^2}
211\f]
