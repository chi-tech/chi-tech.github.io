Detailed Description

Object for controlling log output. There are three levels of verbosity in ChiTech: Zero(Default), One and Two. These can be set on the command line via the switch -v followed by a space and the number for the verbosity (0,1 or 2). The lua command chiLogSetVerbosity(int_level) achieves the same.
