◆ IsRawCellNeighborToPartitionKBA()

bool chi_mesh::VolumeMesherPredefinedUnpartitioned::IsRawCellNeighborToPartitionKBA ( const chi_mesh::UnpartitionedMesh::LightWeightCell &  lwcell)

Determines if a chi_mesh::UnpartitionedMesh::LightWeightCell is a neighbor to the current partition for KBA-style partitioning. This method loops over the faces of the lightweight cell and determines the partition-id of each the neighbors. If the neighbor has a partition id equal to that of the current process then it means this reference cell is a neighbor.
