int chiMonteCarlonCreateSource ( lua_State *  L)

Creates a simple point source at [0 0 0].

Parameters
SolverHandleint Handle to an existing montecarlo solver.
SourceTypeint Source type identifier. See SourceType below.

_

PropertyIndex

MCSrcTypes.BNDRY_SRC
Source on a surface boundary. Expects to be followed by the boundary number.

MCSrcTypes.MATERIAL_SRC
Source from material defined isotropic source. No value follows.

MCSrcTypes.RESID_SRC
Uses a residual source from a field function. Expects to be followed by a field function handle. This will sample the residual.

MCSrcTypes.RESID_SRC_SU
Same as above but will sample the domain uniformly and adjust the weights of each individual particle.

MCSrcTypes.RESID_MOC
Uses a field-function for computing a residual but uses the Method of Characteristics to trace the uncollided portions.

MCSrcTypes.RESID_MOC_SU
Same as above but will sample the domain uniformly and adjust the weights of each individual particle.

Returns
Handle int Handle to the created source.

◆ chiMonteCarlonSetProperty() [1/2]

int chiMonteCarlonSetProperty ( lua_State *  L)

Executes a MonteCarlon solver.

Parameters
SolverHandleint Handle to the montecarlo solver.
PropertyIndexint Code for a specific property.

_

PropertyIndex

MC_NUM_PARTICLES
Number of particles to run. Expects to be followed by an integer specifying the amount of particles to run. Default 1000.

MC_TFC_UPDATE_DIV
Number of divisions of the number of particles to use for tally fluctuation chart (TFC) update. Expects to be followed by an integer specifying the number of bins. Default 10.

MC_MONOENERGETIC
Forces the scattering out of a group to be treated like absorbtion. Expects to be followed by a boolean value. Default false.

MC_SCATTERING_ORDER
Sets the scattering order used for building discrete scattering angles. Expect to be followed by an integer specifying the order. Default 10. Note: when this number is set greater than the scattering order available in the provided cross-sections then the scattering order will default to that available.

MC_FORCE_ISOTROPIC
Flag forcing isotropic scattering. Expects to be followed by a boolean value. Default false.

MC_TALLY_MULTIPLICATION_FACTOR
Classical global tally multiplication factor to be applied after normalization per source particle. Expects to be followed by a float. Default 1.0.

MC_MAKE_PWLD_SOLUTION
Classical global tally multiplication factor to be applied after normalization per source particle. Expects to be followed by a float. Default 1.0.

MC_UNCOLLIDED_ONLY
Trace source particles as uncollided particles, adjusting the weight instead of scattering or absorbing.
