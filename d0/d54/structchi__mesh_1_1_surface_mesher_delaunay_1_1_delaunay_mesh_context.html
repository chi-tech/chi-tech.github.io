Detailed Description

This context assists in three situations. The first is where there are multiple boundaries between regions that requires conformal meshing (connected by edges). The second is where a single boundary comprises a number of non-planar non-contigous surfaces where each non planar surface need to be conformal as well (through edge connections). The third type is interfaces between regions which can be adjoined through edges and/or surfaces.
