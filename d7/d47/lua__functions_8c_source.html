258SLICE_POINT = Reference point for SLICE type FFIs.\n

270###OpTypes
271Basic operations are volume sum, volume average or volume max. The volume
272sum is computed from
273\f[
274Sum = \sum_k \sum_i u_i \int_V N_i .dV.
275\f]
276The volume average is computed from
277\f[
278Avg = \frac{\sum_k \sum_i u_i \int_V N_i .dV}{\sum_k \sum_i \int_V N_i .dV}
279\f]
280The volume max is simply \f$ max_{k,i}(u_i) \f$.\n\n
281OP_SUM\n
282For volume interpolations, computes the volume integral.\n
283\n
284OP_AVG\n
285For volume interpolations, computes the volume average.\n
286OP_MAX\n
287For volume interpolations, computes the volume max.\n
288\n
289A modified version of these operations are also available. Instead of OP_SUM,
290OP_AVG and OP_MAX, the user may supply OP_SUM_LUA, OP_AVG_LUA and OP_MAX_LUA
291which then needs to be followed by a string value `LuaFunctionName` of a lua
292function of the following form:
293\code
294function LuaFunctionName(ff_value, mat_id)
295 ret_val = 0.0; --Or some computation
296 return ret_val
297end
298\endcode
299This code will be called to return a value \f$ f(u_i) \f$ to be used instead of
300the field function \f$ u_i \f$.\n
