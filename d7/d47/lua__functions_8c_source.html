1106  * \f}
1107  *

1109  * using

1118  * assuming one uses the extruder mesher the template boundary would've been
1119  * assigned to index 0 and this will be transfused to all the boundaries in the
1120  * lateral periphery. The top and bottom boundaries are flat for extruded
1121  * geometries and therefore are assigned their own id's. In the extruded case
1122  * the last boundary index is always assigned to the top boundary and
1123  * second-to-last boundary index is always assigned to the bottom boundary. Boundary
1124  * types are specified using the ChiLua::chiDiffusionSetProperty function call,
1125  * using the BOUNDARY_TYPE property index.
1126  *

1128  * associated with cells. The default property mapping is shown below. If
1129  * the property at index [1] (which is mapped to the source value q) is not
