chi_mesh.SnapToPlaneMeshModifier
A Meshing » Mesh Modifiers

Modifier that will snap nodes, that are within a tolerated distance of a plane, to the plane. This modifier is useful for straitening a boundary edge or aligning vertices that misaligned during meshing.

Required Input parameters

type=ARRAY. The normal of the plane to which the nodes are to be snapped.

type=ARRAY. The anchor point of the plane.

Optional Input parameters

type=BOOLEAN. If set to true, only boundary nodes will be snapped.

Default value: true

type=BOOLEAN. If set, only faces that match the plane normal (within tolerance) will have their nodes snapped.

Default value: false

type=FLOAT. Tolerance per dimension within which a face/edge will be aligned

Default value: 1e-05(double)
