72 - `num_faces` <I>int</I>. The value of this field represent the number of faces
73  specified for this cell. Each face is contained in a field "faceX" where X is
74  the face index.
75 - `material_id` <I>int</I>. (Optional) The value of this field holds a material
76  identifier. If not provided, will be defaulted to -1.
77 - `faceX` <I>table</I>. A field holding a lua-table containing the vertex-ids
78  of face X. There must be `num_faces` of these fields, i.e., "face0", "face1", etc.
