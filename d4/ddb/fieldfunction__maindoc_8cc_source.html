1 /** \defgroup LuaFieldFunc Field Functions
2  * \ingroup LuaPhysics
3  *
4  * The concept of a field function is a mature concept in Hydrodynamic
5  * simulation codes such as StarCCM+ and OpenFOAM. So what is it?
6  *
7  * \image html "FieldFunction.png" width=500px
8  *
9  * In the most basic sense a field function is the values of a field as a
10  * function of space and time. This can be as simple as cell average values
11  * located at cell centroids. Such a simple case would be a field function
12  * that is defined by a set of delta functions, therefore the field is only
13  * defined at those specific values via its associated grid (mesh).
14  * In practice however the cell average values
15  * will have do be constant on an entire cell and any position within a given
16  * cell should give this value. Hence the association with a discretization
17  * method. Given a grid and a discretization method the value of a field
18  * function can be obtained without the need to know any information of where
19  * that value is stored or from which physics solver this value was populated.
20  *
21  * This concept will be developed further.
22  * */
