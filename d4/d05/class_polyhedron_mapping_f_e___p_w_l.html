Detailed Description

Object for handling piecewise linear shape functions on polyhedron shaped 3D cells.

This object has a whitepaper associated with it (here)

Some notes on indexing:

  • IntV_shapeI_gradshapeJ, given i and j results in a vector.
  • IntS_shapeI_shapeJ, requires f, then cell i, then cell j
  • face_dof_mappings, is as follows face_dof_mappings[f][fi] and returns cell i.
