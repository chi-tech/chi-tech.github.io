Options available on NonLinearSolver

Note: This object is not constructable

Optional Input parameters

type=FLOAT. Linear absolute tolerance

Default value: 1e-08(double)

type=FLOAT. Linear divergence tolerance

Default value: 1e+06(double)

type=FLOAT. GMRes breakdown tolerance

Default value: 1e+06(double)

type=INTEGER. GMRes restart interval

Default value: 30

type=INTEGER. The maximum allowed non-linear iterations where the linear solver failed to converge.

Default value: 1000

type=INTEGER. Linear maximum iterations

Default value: 100

type=STRING. The linear solver method to use.

Default value: "gmres"

type=FLOAT. Linear relative tolerance

Default value: 1e-08(double)

type=STRING. A name to assign to the solver.

Default value: "NonLinearSolver"

type=FLOAT. Non-linear absolute tolerance

Default value: 1e-08(double)

type=INTEGER. Non-linear maximum iterations

Default value: 50

type=INTEGER. The maximum allowed residual evaluations. Negative number disables this.

Default value: -1

type=STRING. The non-linear method to use.

Default value: "JFNK"

Allowable values: "JFNK", "PJFNK", "NEWTON", "LINEAR"

type=FLOAT. Non-linear relative tolerance

Default value: 1e-08(double)

type=FLOAT. Non-linear solution tolerance

Default value: 1e-50(double)

type=BLOCK. A table of parameters used in the preconditioner.

type=STRING. The type passed to SNESSetType, if PETSc is used. Consult PETSc's documentation for a list of available types.

Default value: "newtonls"
