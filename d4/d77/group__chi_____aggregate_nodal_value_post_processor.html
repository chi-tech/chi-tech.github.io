Required Input parameters

type=BLOCK. Field function handle or name.

type=STRING. Name of the post processor. This name will be used in many places so make sure it's a useful name.

type=STRING. The required operation to be performed.

Allowable values: "max", "min", "avg"

Optional Input parameters

type=ARRAY. List of events at which the post-processor will execute.

Default value: "SolverInitialized", "SolverAdvanced", "SolverExecuted", "ProgramExecuted",

type=BLOCK. An initial value.

type=INTEGER. Handle to a logical_volume.

Default value: 0

type=STRING. Numeric format to use.

Default value: "general"

Allowable values: "fixed", "floating_point", "scientific", "general"

type=ARRAY. List of events at which the post-processor will print. Make sure that these events are also set for the `PostProcessorPrinter` otherwise it wont print.

Default value: "SolverInitialized", "SolverAdvanced", "SolverExecuted", "ProgramExecuted",

type=INTEGER. Number of digits to display after decimal point

Default value: 6

type=STRING. Controls update events to only execute on the relevant solver'sevent calls.

Default value: ""
